Assorted endpoints for autovoice

  • /podcast?voice=VOICE&token=TOKEN&rss=URL_OF_ARTICLE_RSS_FEED
  • /podcastBasedOnFirstFt/:maxResults/:voice?token=TOKEN
  • /podcastBasedOnFirstFt/:maxResults/:voice?token=TOKEN&skipFirstFtUuids=true
  • /mp3?id=ID_OF_MP3_CONTENT
  • /format?text=some words to be processed
    • format
    • /formatArticleForReading/:uuid (edit the url)
  • /snippet.mp3?text=some words to be processed and spoken&voice=voiceId
    • snippet
  • /uuids for specifying individual articles to be included Audio Articles
    • list all uuids
    • add a uuid (edit the url)
    • delete all uuids
  • /content for fetching content
    • fetch RSS feed, return items (edit the url)
    • fetch an article (edit the url)
    • fetch an article, convert to item (edit the url)
    • fetch all articles from individual uuids
  • /content/search
    • search for an article (edit the url)
    • search for the last few firstFT articles (edit the url)
    • search for the last few firstFT articles and the FT articles they in turn mention (edit the url)
  • /podcast
    • /podcast/basedOnFirstFt/1/Amy and any named UUIDs (edit the url)
    • /podcast/basedOnFirstFt/1/Amy and any named UUIDs, with skipFirstFtUuids=true (edit the url)
  • /validate for validating stuff
    • check if url is valid (NB, not that it exists)
